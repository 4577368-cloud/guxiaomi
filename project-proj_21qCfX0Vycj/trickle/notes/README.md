# RSS新闻订阅邮件推送系统

## 项目简介

这是一个自动化的RSS新闻订阅系统，可以从指定的RSS源获取新闻，根据关键词进行筛选，并通过邮件发送到指定邮箱。

## 核心功能

### 1. RSS订阅管理
- 支持多个RSS源
- 实时获取最新新闻
- 可动态添加/删除RSS源

### 2. 关键词筛选
- 自定义关键词
- 智能匹配标题和描述
- 显示匹配的关键词标签

### 3. 邮件推送
- 配置发件邮箱和授权码
- 支持多个收件人
- 定时发送功能
- HTML格式邮件

### 4. 数据管理
- 云端同步配置
- 本地存储备份
- Trickle Database集成

## 技术架构

### 3. 配置邮箱

- 授权码：已配置
- 收件人：4577368@gmail.com（可添加多个）

### 4. 获取新闻
点击"立即获取新闻"按钮，系统会：
1. 从所有RSS源获取最新新闻
2. 根据关键词自动筛选
3. 在预览页面显示结果

### 5. 发送邮件
在新闻预览页面点击"发送邮件"按钮即可发送。

## 注意事项

### 邮件发送限制
由于浏览器安全限制，无法直接通过SMTP发送邮件。建议采用以下方案：
1. 使用Trickle Database保存配置和新闻数据
2. 创建服务端脚本定时读取数据并发送邮件
3. 或使用第三方邮件服务API（如SendGrid、Mailgun）

### 定时任务
定时发送功能需要配合服务端脚本实现，浏览器端无法实现真正的定时任务。
